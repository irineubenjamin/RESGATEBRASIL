Consultar e solicitar a devolução dos Valores a Receber.

O Valores a Receber é um serviço do Consulta Brasil. Você pode consultar se tem dinheiro esquecido ou inesperado em algum banco, consórcio ou outra instituição financeira podendo resgatar o valor.

Desde o dia 28/06/2023 é possível consultar se há valores a receber no sistema financeiro. A janela de saque se encerra daqui a 15 dias corridos, então essa é a sua última janela de chance de resgatar.

O passo a passo para receber a devolução se encontra abaixo. Siga atentamente os passos para descobrir se há valores disponíveis e caso tenha, realizar o seu saque.

Lembre-se que esse é o único canal autorizado a realizar este procedimento.

Veja como realizar a sua Consulta:

1º PASSO:

Insira seus dados para realizar a Consulta Gratuita e saber a quantia exata que você tem disponível para saque.

2º PASSO:

Aguarde a Consulta ser realizada. O tempo de espera varia entre 1-3 minutos.

3º PASSO:

Basta realizar o saque do valor disponível para você.
